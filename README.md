# Sistema Web de Gestión de Incidencias Informáticas

Este proyecto es una **aplicación web para la gestión de incidencias informáticas** en centros educativos.  
Permite a usuarios, técnicos y administradores registrar, gestionar y analizar incidencias en tiempo real.

## Características principales

- Registro y login de usuarios con roles: `usuario`, `técnico`, `administrador`.
- Creación y seguimiento de incidencias por parte de los usuarios.
- Panel de técnicos para responder incidencias y marcar como resueltas.
- Panel de administradores con gestión de usuarios y roles.
- Sistema de notificaciones en tiempo real y por correo electrónico.
- Gráficas de incidencias activas y resueltas (Chart.js).
- Código limpio y modular en PHP, compatible con MySQL.

## Tecnologías utilizadas

- PHP puro
- MySQL
- HTML5, CSS3, JavaScript
- PHPMailer para envío de correos
- Chart.js para gráficas
